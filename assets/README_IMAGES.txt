# Необходимые изображения для бонусов

Для правильной работы игры необходимо создать следующие изображения в формате PNG:

1. shield.png - изображение щита круглой формы, синего цвета с прозрачным центром
   - Рекомендуемый размер: 64x64 пикселей
   - Используется для бонуса "Щит", который защищает от ударов астероидов

2. speed.png - изображение стрелок или молнии
   - Рекомендуемый размер: 64x64 пикселей
   - Используется для бонуса "Ускорение", который увеличивает скорость корабля

3. magnet.png - изображение магнита
   - Рекомендуемый размер: 64x64 пикселей
   - Используется для бонуса "Магнит", который притягивает топливо и сердечки

4. double_score.png - изображение "x2" или другого символа удвоения
   - Рекомендуемый размер: 64x64 пикселей
   - Используется для бонуса "Двойные очки", который удваивает получаемые очки

5. heart.png - изображение сердца (уже должно существовать)
   - Используется для восстановления жизней

Для создания изображений можно использовать любой графический редактор, поддерживающий формат PNG с прозрачностью.
